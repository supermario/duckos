DuckOS

A tiny (~20mb) PaaS

Builds

ARM11
(ARMv6)

i386
(32-bit)

amd64
(64-bit)

PowerPC

Languages

Node.js Python Ruby PHP Perl

Install

Run the following in a terminal

curl duckos.io/install | sh

Build

Insert an SD card on host and run duck build 

---> Downloading DuckOS...
Select disk device (data will be destroyed): /dev/disk2
---> Writing partitions...
---> Installing files...
Configure root password (duckos):
===> DuckOS install complete!

Find

Insert SD into Pi, connect network cable and turn on.

After a few seconds run duck find

---> Searching for DuckOS hosts...
10.0.1.15 (raspberrypi)

Deploy

$ echo 'Hello world!' > app.js
$ git init; git add app.js
$ git commit -m "First release"
$ git push duck@10.0.1.15 master
---> Node.js app detected...
===> Application deployed
http://10.0.1.15
