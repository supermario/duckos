DuckOS

A tiny (~20mb) OS
for heroku-like code deployments

Builds

RaspberryPi (armv6) i386 (32-bit) amd64 (64-bit)

Languages

Node.js Python Ruby PHP

Install

Insert an SD card on host and run

curl duckos.io/install | sh

---> Installing duck to /usr/local/bin/duck
---> Downloading DuckOS...
Configure root password (duckos):
Select disk device (data will be destroyed): /dev/disk2
---> Writing partitions...
---> Installing files...
===> DuckOS install complete!

Deploy

Insert SD into Pi, connect network cable. Allow 5-10 seconds for device to boot. 

$ duck list
10.0.1.15 (raspberrypi)

Deploy an application

$ echo 'Hello world!' > app.js
$ git init; git add app.js
$ git commit -m "First release"
$ git push duck@10.0.1.15 master
---> Node.js app detected...
===> Application deployed
     http://10.0.1.15
